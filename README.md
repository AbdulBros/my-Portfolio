# Futuristic Developer Portfolio

A modern, responsive developer portfolio built with React, 
featuring a dark futuristic UI theme and AI-accelerated 
design-to-code workflow.

## Tech Stack
React.js, JavaScript, CSS3, HTML5

## Key Features
- Fully responsive design across all device sizes
- Futuristic dark-mode UI with smooth animations
- AI-assisted component generation and layout structuring

## Build Process
Built using an AI-assisted design-to-code workflow — 
prompted UI layout, component structure, and styling, 
then refined and deployed manually.
